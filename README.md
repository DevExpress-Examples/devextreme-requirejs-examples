# DevExtreme with RequireJS

This repository contains an example that shows how to use [RequireJS](http://requirejs.org/) with the [jQuery](http://jquery.com/) approach. The example is an application with a [dxButton](https://js.devexpress.com/Documentation/ApiReference/UI_Components/dxButton/) widget. When you click this button, an [alert dialog](https://js.devexpress.com/Documentation/ApiReference/Common/Utils/ui/dialog/#alertmessageHtml_title) appears. The application requires Node.js and npm to install the [DevExtreme](http://js.devexpress.com/) package and its dependencies.

Used Technologies:      | Example Name:
----------------------- | -------------------------------------
RequireJS with jQuery   | [requirejs-jquery](requirejs-jquery/)
